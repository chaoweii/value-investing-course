## Scenarios are different worlds

A bear case is not “base case minus 20%.” It should describe a coherent failure mechanism:
weaker demand, lower pricing, higher expansion cost, slower operating leverage, or dilution.
The bull case should require observable evidence before receiving a higher probability.

Sensitivity analysis ranks which assumptions carry the valuation. Reverse DCF asks what revenue,
margin, or cash-flow path makes today’s price mathematically reasonable. It does not prove that
the implied path will occur.
