## Purpose

Phase 2 turns business understanding into a price-dependent investment decision.
Margins become outputs of revenue and cost drivers, not optimistic assumptions.

| Lecture | Central question | Practical output |
|---|---|---|
| Revenue-and-cost drivers | What causes each line to change? | Dependency-aware forecast |
| ROIC and reinvestment | How fast can intrinsic value compound? | Reinvestment runway |
| DCF, SOTP, and assets | Which valuation method matches the economics? | Valuation triangulation |
| Scenarios and IRR | What is priced in, and what return follows? | Bear/base/bull return map |
| Audit and sizing | How much capital does the uncertainty deserve? | Add/hold/trim/exit rules |

## The governing discipline

The base case must be plausible without requiring every favorable assumption to
occur simultaneously. The bear case must describe a real failure mechanism. The
bull case must identify evidence required before receiving capital.
