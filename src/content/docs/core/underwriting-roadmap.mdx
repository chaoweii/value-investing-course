## Purpose

Phase 2 turns business understanding into a price-dependent investment decision. A DCF is not
a formula for producing a target price. It is a structured argument about how customers, costs,
reinvestment, risk, and capital allocation become owner value per share.

## Governing discipline

The base case must be plausible without requiring every favorable assumption simultaneously.
The bear case must describe a real failure mechanism. The bull case must identify evidence
required before receiving capital. Margins are outputs, and terminal value is never a hiding place.
